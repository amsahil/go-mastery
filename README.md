# 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/go-mastery.git
```

Go into the project:

```bash
cd go-mastery
```

Start local server:

```bash
python3 -m http.server 8080
```

Open browser:

```bash
http://localhost:8080
```

---

# 🌍 Deployment

This project is hosted using GitHub Pages.

Deployment steps:

1. Push project to GitHub
2. Open:
   Settings → Pages
3. Select:

   * Branch: `main`
   * Folder: `/root`
4. Save

GitHub automatically publishes the website.
